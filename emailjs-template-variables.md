# EmailJS template variabelen

Gebruik deze variabelen in je EmailJS template:

- `{{name}}` - naam van de klant
- `{{email}}` - e-mailadres van de klant
- `{{project_type}}` - wat de klant wil laten bouwen
- `{{message}}` - barterwaarde / toelichting van de klant
- `{{website_url}}` - link naar je website, bijvoorbeeld `https://randyb2024.github.io/...`
- `{{logo_url}}` - publieke URL naar het logo, bijvoorbeeld `https://randyb2024.github.io/jouw-repo/barterweb-logo.svg`

Onderwerpregel:

```text
Je BarterWeb aanvraag is ontvangen, {{name}}
```

# BarterWeb landingpage

Moderne futuristische BarterWeb landingpage met animaties, glassmorphism, een value-first flow en zonder testimonials/reviews.

## Bestanden

- `index.html` - pagina-opbouw en content
- `styles.css` - volledige responsive styling
- `script.js` - smooth scroll, scroll-reveal animaties en formulierfeedback

## Lokaal openen

Open `index.html` direct in je browser, of serveer de map met een simpele static server.

## Verder bouwen in GitHub

Deze versie heeft geen build tools of dependencies nodig. Je kunt hem later makkelijk omzetten naar React, Next.js of een ander framework door de secties uit `index.html` als componenten over te nemen.
